StreamsAccess

public protocol StreamsAccess : AnyObject

The interface to access everything related to the streams

  • streams

    The list of currently available streams

    Declaration

    Swift

    var streams: [MycrocastStream] { get }
  • streams$

    Publisher that contains the current list and will publish the list again anytime an update happens either add, update or remove

    Declaration

    Swift

    var streams$: AnyPublisher<[MycrocastStream], Never> { get }
